# ADP Meetup Webpack 5
Presentation about Webpack 5 from ADP Meetup

### [Link to Presentation](https://sw1tchdev.github.io/adp-meetup-webpack-presentation/)

## Install
```shell
# install deps
npm i
# or
yarn
```
## Usage
### Start
```shell
npm run start
# or
yarn run start
```
### Build
```shell
npm run build
# or
yarn run build
```
